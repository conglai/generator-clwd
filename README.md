## 安装
```
~ npm i yeoman -g
~ npm i generator-clwd -g
```

## 使用
```
~ yo clwd:rc # React 组件
~ yo clwd
```

## 功能说明

### rc生成器
React Component，开发环境包括:

* gulp
* webpack
* babel
* browser-sync
* eslint
